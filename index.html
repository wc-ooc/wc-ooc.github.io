04 — REFLECTIONS

作品心得

每个项目都是一次思维方式的迭代。以下是我在创作过程中沉淀的一些感悟与反思。

McDonald's UI
完整应用系统

这个项目让我深刻理解了「系统化设计」的重要性——通过明确的组件规范、色彩体系、交互逻辑，使整个应用保持高度统一，提升用户操作效率和视觉舒适度。

今日运势
动画交互体验

这个作品让我思考「氛围感」如何通过颜色、光影与动效节奏共同营造——占卜结果揭晓的那一刻，多层渐变、缓动曲线、glow 效果协同，将抽象的「运势」转化为感知得到的「惊喜」。

Telegram Chat
即时通讯 UI

通过复现 Telegram 的设计语言，我体会到了「信息架构」对用户体验的深刻影响——清晰的对话列表、精确的消息状态反馈、直观的交互逻辑，让复杂的通讯功能变得自然而流畅。

OVERALL REFLECTION
"做这些项目让我真正理解了：代码是工具，设计是语言，而用户体验才是最终想说的那句话。"
从一开始只想着「功能实现就好」，到后来开始追问「用户为什么要这样操作」——这种思维转变是这一阶段最宝贵的收获。技术学习是有时效性的，但对用户心理与交互逻辑的洞察却是永恒的资产。
— 刘旭，2025
