04 — REFLECTIONS

作品心得

这三个作品分别对应通讯工具、情绪化互动与移动端点餐场景。它们让我更清楚地理解：不同产品类型需要不同的体验重心。

Telegram 聊天界面
Chat UI Prototype

这个作品让我意识到，聊天界面的重点不是装饰，而是信息层级与阅读效率。头像、未读数、时间、在线状态和气泡间距都要足够清晰，用户才能在大量消息里快速判断重点。

今日运势
Zodiac Fortune

这个作品更偏情绪化体验，功能很简单，但氛围很重要。星空、极光、抽签滚动和结果光效共同服务于“期待感”，让我理解到动效节奏可以直接影响用户的情绪反馈。

麦当劳点餐 UI
Mobile Ordering System

这个作品让我更关注商业产品的转化路径：搜索、活动 Banner、分类筛选、加购和购物车需要形成连续动线。好的点餐界面不是把内容堆满，而是让用户更快看到想买的东西并完成下单。

OVERALL REFLECTION
"不同作品需要不同的设计判断：聊天界面追求清晰，运势卡片追求情绪，点餐系统追求效率。"
这三个项目让我从不同角度理解了 UI 设计：信息型界面要减少干扰，互动型界面要制造记忆点，交易型界面要缩短决策路径。相比只完成页面，我更希望每个作品都能说明自己为什么这样布局、为什么这样反馈、为什么这样引导用户继续操作。
— 刘旭，2025
